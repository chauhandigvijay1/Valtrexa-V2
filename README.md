# 📖 What is Valtrexa?

Most job seekers today use a fragmented workflow:

* Resume Builder
* Job Board
* Spreadsheet
* Notes App
* Recruiter Tracker
* Interview Notes
* Follow-Up Reminders
* Email Inbox

Every step exists in a different tool.

Valtrexa is designed to unify the entire career workflow into a single platform.

Instead of managing disconnected tools, candidates operate through one intelligence layer that continuously learns from resumes, projects, applications, recruiter interactions, company research, and interview history.

Valtrexa is being built as an **Autonomous Career Operating System**.

---

# 🎯 Vision

The long-term vision is simple:

> Give every candidate an operating system that manages their career as intelligently as modern CRMs manage customers.

Valtrexa is not a job board.

Valtrexa is not a resume builder.

Valtrexa is not an ATS.

Valtrexa is a career intelligence platform designed to orchestrate the entire journey from opportunity discovery to interview success.

# 🧠 Candidate Brain

Candidate Brain acts as the single source of truth.

Instead of repeatedly entering the same information across forms, applications, resumes, and outreach campaigns, Valtrexa centralizes everything into one intelligence layer.

## Stores

### Identity

* Name
* Email
* Phone
* Location

### Career Information

* Current Role
* Current Company
* Experience
* Salary Expectations
* Career Goals
* Communication Style

### Professional Assets

* Resume
* Skills
* Projects
* Education
* Certifications
* Portfolio
* GitHub
* LinkedIn

### Preferences

* Preferred Roles
* Preferred Locations
* Remote Preferences

## Workflow

```text
Resume Upload
        ↓
Resume Parsing
        ↓
Candidate Brain Sync
        ↓
Profile Auto Population
```

Users should never manually re-enter information already present in uploaded resumes.

# 🎯 Opportunity Radar

The Opportunity Radar evaluates opportunities using multiple scoring systems.

## Inputs

* Match Score
* Freshness Score
* Company Quality Score
* Strategic Value Score

## Classification

| Tier   | Action            |
| ------ | ----------------- |
| Tier A | Auto Apply        |
| Tier B | Auto Apply        |
| Tier C | Approval Required |
| Tier D | Ignore            |

This ensures attention is focused on high-value opportunities.
